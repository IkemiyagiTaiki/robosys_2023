## ダウンロード方法

Githubを利用できる環境で以下のコマンドをコピーする

```

$ git clone  https://github.com/IkemiyagiTaiki/robosys_2023.git


```
その後、実行権限を付与する。

```

$ chmod +x plus

```


## plus

標準出力から読み込んだ数字を足す

### 使用方法

```


$ seq 5 | ./plus
15


```


## sqrt

標準出力からよみこんだ数字を平方根する

## 使用方法

```

$ ./sqrt 4
2

```
